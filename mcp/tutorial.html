Thread · Asistente de tareas

Manejá tus tareas hablando con Claude

Conectá Thread a Claude una sola vez y después pedile las cosas en palabras: ver qué tenés para hoy, empezar una tarea, completarla o mandarla a revisión. Sin abrir la app.

Antes de empezar — necesitás

  • Node.js 20+ instalado. Verificá con node -v.
  • El repo de Thread en tu compu (te lo pasa quien administra el proyecto).
  • Claude Code (recomendado) o Claude Desktop.
  • Tu email y contraseña de Thread — los mismos con los que entrás a la app.

Puesta en marcha — 4 pasos

1

Traé el proyecto e instalá

Abrí una terminal en la carpeta del repo y corré:

# dentro de la carpeta del proyecto Thread
git pull
npm install

npm install baja todo, incluido el conector (MCP). Una sola vez.

2

Poné tu login de Thread

Creá un archivo llamado .env.local en la raíz del proyecto con tus datos:

THREAD_EMAIL=vos@equipo.com
THREAD_PASSWORD=tu-contraseña
Privado. Ese archivo queda solo en tu compu y no se sube al repo. Son tus credenciales personales: Claude ve exactamente lo que verías vos en la app.
3

Conectalo a Claude

Claude Code: nada que hacer. Al abrir la carpeta del proyecto, Claude detecta el conector thread y te pregunta si lo aprobás. Aceptá y listo.

Claude Desktop: Ajustes → Developer → Edit Config, y agregá el bloque (cambiá la ruta por la de tu carpeta):

{
  "mcpServers": {
    "thread": {
      "command": "node",
      "args": ["C:/ruta/a/Thread/mcp/server.mjs"]
    }
  }
}

Guardá y reiniciá Claude Desktop.

4

Probá que anda

Escribile a Claude:

¿qué tareas tengo para hoy?
Si te devuelve tu lista de tareas, quedó conectado. Si dice que faltan THREAD_EMAIL / THREAD_PASSWORD, revisá el paso 2.

Qué le podés pedir

¿qué tengo pendiente esta semana?

Lista tus tareas ordenadas por vencimiento.

empezá la tarea del login

La pasa a «En curso».

marcá «diseñar navbar» como hecha

La completa.

mandá la API de pagos a revisión

Avisa al responsable de revisión.

creá una tarea «arreglar footer» en el proyecto Web para mí

La crea y te la asigna.

contame de qué se trata «exportar PDF»

Muestra el detalle y los comentarios.

Podés nombrar la tarea por su título (aunque sea parte) — si hay varias parecidas, Claude te muestra las opciones para que elijas.

Si algo no anda

  • «Faltan THREAD_EMAIL / THREAD_PASSWORD» → el archivo .env.local no existe o está mal escrito. Revisá el paso 2.
  • «Login falló» → email o contraseña incorrectos. Son los mismos con los que entrás a la app de Thread.
  • «No hay persona registrada con ese email» → tu usuario todavía no está cargado en el equipo. Avisale a quien administra el proyecto.
  • Claude Code no muestra el conector → asegurate de abrir la carpeta del proyecto (donde está .mcp.json) y de haber corrido npm install.
Thread · conector MCP. Todo lo que hacés desde Claude respeta tus permisos: es lo mismo que harías dentro de la app.
